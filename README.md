## 📱 Как использовать

<details>
<summary><b>📱 Hiddify (Android / iOS)</b></summary>

1. Скачай [Hiddify](https://hiddify.com/) из Google Play / App Store
2. Открой приложение
3. Нажми **"+"** → **"Add from URL"**
4. Вставь ссылку подписки
5. Нажми **"Add"**
6. Готово! ✅

</details>

<details>
<summary><b>💻 Hiddify (Windows / macOS)</b></summary>

1. Скачай [Hiddify Desktop](https://github.com/hiddify/hiddify-next/releases)
2. Открой приложение
3. **Profiles** → **Add Profile** → **From URL**
4. Вставь ссылку подписки
5. Сохрани
6. Готово! ✅

</details>

<details>
<summary><b>🤖 V2rayNG (Android)</b></summary>

1. Скачай [V2rayNG](https://github.com/2dust/v2rayNG/releases)
2. Открой приложение
3. Нажми **"+"** → **"Import config from URL"**
4. Вставь ссылку подписки
5. Готово! ✅

</details>

<details>
<summary><b>🍎 Shadowrocket (iOS)</b></summary>

1. Открой Shadowrocket
2. Нажми **"+"** в правом верхнем углу
3. **Type** → **Subscribe**
4. Вставь ссылку подписки
5. Готово! ✅

</details>

## 🔄 Автообновление

Подписка обновляется **автоматически каждые 5 минут** через GitHub Actions.

Твое VPN приложение будет автоматически подтягивать новые конфиги при обновлении подписки.

Для ручного обновления:
- **Hiddify:** Нажми на профиль → **Update**
- **V2rayNG:** Потяни вниз список конфигов
- **Shadowrocket:** Нажми на подписку → **Update**

## ⚙️ Источники конфигов

Конфиги агрегируются из проверенных источников:

| Источник | Протоколы | Описание |
|----------|-----------|----------|
| [igareck/vpn-configs-for-russia](https://github.com/igareck/vpn-configs-for-russia) | VLESS, SS, VMess, Trojan | Основной источник для России |
| [mahdibland/V2RayAggregator](https://github.com/mahdibland/V2RayAggregator) | Все протоколы | Агрегатор со всего мира |
| [yebekhe/TelegramV2rayCollector](https://github.com/yebekhe/TelegramV2rayCollector) | Все протоколы | Коллектор из Telegram каналов |
| [barry-far/V2ray-Configs](https://github.com/barry-far/V2ray-Configs) | Все протоколы | Проверенные конфиги |
| [peasoft/NoMoreWalls](https://github.com/peasoft/NoMoreWalls) | Все протоколы | Антицензурные конфиги |
