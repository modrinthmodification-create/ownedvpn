## 📱 Как добавить в Hiddify

### Android / iOS
1. Открой **Hiddify App**
2. Нажми **"+"** → **"Add from URL"**
3. Вставь одну из ссылок выше
4. Готово! ✅

### Windows / macOS
1. Открой **Hiddify**
2. **Profiles** → **Add Profile** → **From URL**
3. Вставь ссылку
4. Сохрани

### Обновление конфигов
Hiddify автоматически обновляет подписки. Для ручного обновления:
- Нажми на профиль → **Update** или **Refresh**
