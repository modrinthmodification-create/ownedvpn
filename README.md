## 📱 Как использовать

### В Hiddify App (Android/iOS/Windows/macOS)

1. Открой Hiddify App
2. Нажми на "+" или "Добавить профиль"
3. Выбери "Добавить из URL" или "Subscription URL"
4. Вставь ссылку выше
5. Нажми "Добавить" или "Subscribe"

### В Hiddify Next

1. Открой приложение
2. Перейди в раздел "Profiles" или "Профили"
3. Нажми "Add Profile" → "From URL"
4. Вставь ссылку
5. Сохрани
