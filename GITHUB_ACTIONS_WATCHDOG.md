# VPN Watchdog на GitHub Actions

## Как это работает

✅ **Полностью бесплатно** - работает на GitHub Actions
✅ **24/7 мониторинг** - проверка каждые 5 минут
✅ **Автоматическое обновление** - скачивает свежие конфиги
✅ **Не требует сервера** - все работает в облаке GitHub
✅ **Автокоммит** - обновленные конфиги сразу в репозиторий

## Установка

1. Скопируй файлы в свой репозиторий:
```
.github/workflows/vpn-watchdog.yml
.github/workflows/watchdog_action.py
```

2. Закоммить и запушить:
```bash
cd C:\scripts\vpn-subscription
git add .github/
git commit -m "Add VPN watchdog on GitHub Actions"
git push
```

3. Готово! GitHub Actions автоматически запустится

## Проверка работы

1. Открой свой репозиторий на GitHub
2. Перейди в **Actions**
3. Увидишь "VPN Watchdog" - он запускается каждые 5 минут
4. Можешь запустить вручную через "Run workflow"

## Что делает

Каждые 5 минут:
1. Проверяет интернет
2. Проверяет доступность VPN конфигов
3. Скачивает свежие конфиги из источника
4. Если конфиги изменились - автоматически коммитит в репозиторий
5. Твоя подписка в Hiddify обновляется автоматически

## Логи

Все логи доступны в GitHub Actions:
- Открой репозиторий → Actions
- Кликни на любой запуск "VPN Watchdog"
- Увидишь полные логи работы

## Преимущества

✅ Работает даже когда твой ПК выключен
✅ Не нужен VPS сервер
✅ Полностью бесплатно
✅ Автоматические обновления конфигов
✅ История всех изменений в git

## Ограничения GitHub Actions

- Запуск каждые 5 минут (минимум для cron)
- 2000 минут в месяц бесплатно (хватит на весь месяц)
- Если нужно чаще - можно уменьшить до */3 (каждые 3 минуты)
